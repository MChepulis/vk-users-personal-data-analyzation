# vk-users-personal-data-analyzation

## How to run
run bash scripts from project root directory

example: 


```bash
bin/start_connector.sh
bin/run_task.sh
```


## Requirements
 Python 3.6x

Libraries:
* elasticsearch
* elasticsearch-dsl
* vk-api
* matplotlib-pyplot
* geopy
* plotly

Technologies
* elasticsearch db
